# VELOCITY ARTIFACTS

A high-end, conceptual footwear store built with React, Tailwind CSS, and Framer Motion.

## Aesthetic: "Kinetic Editorial"
This project avoids the typical "clean" e-commerce look in favor of a brutalist, technical, and editorial aesthetic.

### Key Design Features:
- **Typography**: Uses *Playfair Display* for a luxury editorial feel paired with *JetBrains Mono* for a technical/industrial vibe.
- **Palette**: Deep Charcoal (#0A0A0A) background with high-contrast Bone (#F5F5F0) text and sharp Safety Orange (#FF3E00) accents.
- **Interactions**:
    - Smooth scroll-reveals and staggered entrance animations.
    - Kinetic hover effects on product cards.
    - Glassmorphism and technical grid backgrounds.
    - Fully functional cart system with state management.
- **Responsive**: Mobile-first design that scales beautifully to high-resolution displays.

## Tech Stack
- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Setup
1. `npm install`
2. `npm run dev`
3. `npm run build` (to check production readiness)
